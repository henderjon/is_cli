# is_cli

Defines a constant for checking if the current process is being run via CLI.

# usage

`if(IS_CLI){ /* noop */ }` or `if(is_cli()){ /* noop */ }`

# installation

Using [composer](http://getcomposer.org/) `"require" : { "henderjon/is_cli": "~1.0" }`
